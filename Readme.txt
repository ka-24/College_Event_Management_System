There are two roles in the system: Admin and User.
Users can:
* Search Events.
* Join Events.
* View Booking events.
* Change password.
* Edit their own profile.
Admin can:
* Create, Retrieve, Update, Delete Events.
* Retrieve, Update, Delete Users events joined.
* Search perticular Event Joined by user.
* Add events in booking details , remove events from booking details.
* Edit their own profile.

Built With
HTML, CSS, JavaScript, Mysql and PHP.

Installation Process
* Download and install XAMPP/WAMP/LAMP server and download the files of this project
* Copy the folder of this project in your xampp/htdocs/www folder
* Start XAMMP. Start Apache and SQL server. Go to phpmyadmin and create a new database named 'ems'
* Go to the ems database created and click on 'Import' option in the top menu
* Upload the ems.sql file and import it
* Open your web browser and check if you got the website running on your localhost (http://localhost/(Mini_Project file name)/)
